Data Structures

struct  Elevator
 Keep track of variables that are important for operating the elevators behaviour in fsm.
orderArray contains the value of all order buttons: 1 if a given button has been pressed, 0 if not.
First 4 rows corresponds to cabin buttons, 4 last to buttons outside of cabin.
First coloumn corresponds to down buttons, second (and last) to up buttons.
currentObstructionValue keep track of value of obstruction button. 1 if pressed, 0 if not.
lastObstructionValue keep track of last value of obstruction button. 1 if pressed, 0 if not.
Used to check if obstruction button recently has changed value.
doorOpen 1 if door is open, 0 if not.
currentFloor keep track of which floot the elevator is in. Is set to the floor the elevator last was in, if elevator is between floors.
currentDirection keep track of which direction the elevator is moving; up/down/still.
lastDirection keep track of which direction the elevator last was moving; up/down/still.
Used for state transitions and deciding next moving direction in fsm.
